# MQTT Security Tester

This project is a tool to test the security of MQTT brokers. It can be used to check for common vulnerabilities, such as:

* Weak authentication
* Lack of encryption
* Susceptibility to denial-of-service attacks

## Usage

To use the MQTT Security Tester, you will need to have Python 3 installed. You can then install the dependencies by running the following command:

```
pip install -r requirements.txt
```

Once the dependencies are installed, you can run the MQTT Security Tester by running the following command:

```
python mqtt_security_tester/main.py
```
